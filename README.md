# FrotaFlow - Sistema de Gestão de Manutenção de Frota de Ônibus

## Sobre o Projeto

O FrotaFlow é um sistema web desenvolvido com o objetivo de simplificar a gestão de manutenção de frotas de ônibus. A aplicação oferece uma interface intuitiva e de fácil aprendizado, permitindo o controle eficiente de veículos, ordens de serviço e estoque de peças.

## Funcionalidades

### Módulo de Manutenção
- Listagem de ordens de serviço
- Filtro por placa e status
- Criação, edição e exclusão de ordens
- Acompanhamento do status da manutenção

### Cadastro de Ônibus
- Registro completo de veículos (placa, modelo, montadora, ano, km, chassi)
- Upload de imagem do veículo
- Edição e exclusão de registros

### Almoxarifado
- Controle de estoque de peças e materiais
- Categorização de itens
- Monitoramento de estoque mínimo
- Cadastro e edição de itens

### Relatórios
- Geração de relatórios gerenciais (em desenvolvimento)

### Autenticação
- Login de usuários
- Cadastro de novos usuários

## Tecnologias Utilizadas
